# Resupply Loop

A minimal Next.js 14 app for stablecoin leverage loop strategies.

## Getting Started

```bash
npm install && npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Features

- Landing page with 3-step explainer
- Loop interface with amount input and leverage slider (1-4x)
- Real-time preview of position size, net APY, and health factor
- Dark theme with Tailwind CSS

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
